    Project maintained by jurassicplayer Hosted on GitHub Pages — Theme by mattgraham

    Project Description:

    This project has a variety of themes available for your use. At times, it will have some placeholder themes (why are gtk themes so hard to understand) as I learn how to make them in my spare time. The end goal is to have a complete suite of uniform themes for most, if not all, of the programs that I use.

    Installation:

    Clone the git repository to your local system and then copy them to the correct locations. There is also an optional script that copies everything to the correct locations in /usr, thus requires sudo permissions.

    $ git clone https://github.com/jurassicplayer/Weeb-Themes.git
$ cd Weeb-Themes
$ sudo ./install.sh
